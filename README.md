# Bluehill.SdkStyleTemplates
SDK-style templates for .NET Framework

## How to install
Run the following command in the console:

```
dotnet new install Bluehill.SdkStyleTemplates
```

## How to uninstall
Run the following command in the console:

```
dotnet new uninstall Bluehill.SdkStyleTemplates
```
